# 📸 WordPress 後台建立 Banner 和 Product 的圖解教學

## 🎯 重要觀念

**Banner 和 Product 都是使用「一般文章」，只是透過「分類」來區分！**

---

## 📋 步驟 1：建立分類

### 1.1 前往分類管理
1. 登入 WordPress 後台：`http://localhost/headless/wp-admin`
   - 帳號：`admin`
   - 密碼：`admin123`

2. 在左側選單找到：**文章 → 分類**
   ```
   左側選單：
   ├── 控制台
   ├── 文章 ← 點這裡
   │   ├── 所有文章
   │   ├── 新增文章
   │   ├── 分類 ← 再點這裡
   │   └── 標籤
   ```

### 1.2 建立 Banner 分類
在「新增分類」表單中填寫：

```
┌─────────────────────────────────┐
│ 名稱：Banner                     │
│ 代稱：banner                     │ ← 必須是小寫
│ 描述：Banner 輪播分類（選填）     │
└─────────────────────────────────┘
```

點擊「新增分類」按鈕

### 1.3 建立 Products 分類
重複上面的步驟，建立第二個分類：

```
┌─────────────────────────────────┐
│ 名稱：Products                   │
│ 代稱：products                   │ ← 必須是小寫
│ 描述：產品展示分類（選填）        │
└─────────────────────────────────┘
```

點擊「新增分類」按鈕

✅ **完成後，您應該會在右側看到兩個新分類：Banner 和 Products**

---

## 📝 步驟 2：建立 Banner 文章

### 2.1 前往新增文章
1. 在左側選單點擊：**文章 → 新增文章**
   ```
   左側選單：
   ├── 文章 ← 點這裡
   │   ├── 所有文章
   │   ├── 新增文章 ← 再點這裡
   ```

### 2.2 填寫 Banner 內容

```
┌─────────────────────────────────────────────┐
│ 新增標題                                     │
│ ┌─────────────────────────────────────────┐ │
│ │ 沙發必須與眾不同                         │ │ ← 這會是 Banner 的主標題
│ └─────────────────────────────────────────┘ │
│                                              │
│ 在此輸入內容...                              │
│ ┌─────────────────────────────────────────┐ │
│ │ 義式輕奢生活，從一張沙發開始              │ │ ← 這是內容（可選填）
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 2.3 設定分類（右側欄位）

在右側找到「分類」區塊：

```
┌─────────────────────────┐
│ 分類                     │
├─────────────────────────┤
│ ☐ 未分類                 │
│ ☑ Banner  ← 勾選這個！   │
│ ☐ Products               │
└─────────────────────────┘
```

**重要：一定要勾選 Banner 分類！**

### 2.4 設定特色圖片（右側欄位）

在右側找到「特色圖片」區塊：

```
┌─────────────────────────┐
│ 特色圖片                 │
├─────────────────────────┤
│ [設定特色圖片] ← 點這裡   │
└─────────────────────────┘
```

點擊後會開啟媒體庫：
1. 點擊「上傳檔案」
2. 選擇您的沙發背景圖片（建議 1920x1080px）
3. 點擊「設定特色圖片」

**重要：特色圖片會成為 Banner 的背景圖！**

### 2.5 發布文章

點擊右上角的「發佈」按鈕

```
┌─────────────────┐
│   [發佈]         │ ← 點這裡
└─────────────────┘
```

✅ **第一個 Banner 完成！**

### 2.6 建立更多 Banner（選用）

重複步驟 2.1-2.5，建立 2-3 個 Banner，例如：
- Banner 2：義式輕奢生活
- Banner 3：品味生活從此開始

## 🛋️ 步驟 3：建立 Product 文章

### 3.1 前往新增文章
同樣點擊：**文章 → 新增文章**

### 3.2 填寫 Product 內容

```
┌─────────────────────────────────────────────┐
│ 新增標題                                     │
│ ┌─────────────────────────────────────────┐ │
│ │ 義式家居設計哲學                         │ │ ← Product 標題
│ └─────────────────────────────────────────┘ │
│                                              │
│ 在此輸入內容...                              │
│ ┌─────────────────────────────────────────┐ │
│ │ 21年家具生產製造設計經驗積澱，           │ │
│ │ 摸索推出義式輕奢品牌——德客·皮沙發，      │ │ ← Product 描述
│ │ 專為3億新中產量身打造。                   │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 3.3 設定分類（右側欄位）

```
┌─────────────────────────┐
│ 分類                     │
├─────────────────────────┤
│ ☐ 未分類                 │
│ ☐ Banner                 │
│ ☑ Products ← 勾選這個！  │
└─────────────────────────┘
```

**重要：一定要勾選 Products 分類！**

### 3.4 設定特色圖片

上傳沙發產品圖片（建議 1000x600px）

### 3.5 發布文章

點擊「發佈」

✅ **第一個 Product 完成！**

### 3.6 建立更多 Product（建議）

重複步驟 3.1-3.5，建立 3-5 個產品，例如：
- Product 2：羅馬藝術經典傳承
- Product 3：地中海熱情風尚
- Product 4：美式現代舒適主義
- Product 5：北美自然純淨美學

## 🔍 步驟 4：檢查是否成功

### 4.1 查看 Banner 文章
1. 前往：**文章 → 所有文章**
2. 在上方篩選器選擇「Banner」分類
3. 應該會看到您建立的 Banner 文章

### 4.2 查看 Product 文章
1. 前往：**文章 → 所有文章**
2. 在上方篩選器選擇「Products」分類
3. 應該會看到您建立的 Product 文章

### 4.3 測試 API
在瀏覽器開啟：
- Banner API: `http://localhost/headless/wp-json/wp/v2/posts?categories=1&_embed`
- Product API: `http://localhost/headless/wp-json/wp/v2/posts?categories=2&_embed`

如果看到 JSON 資料，表示成功！

### 4.4 查看前端效果
開啟：`http://localhost:3000`

應該會看到：
- ✅ Hero Banner 顯示您的 Banner 文章
- ✅ Orbit Section 顯示您的 Product 文章
- ✅ 可以用左右箭頭切換

---

## 📌 重點提醒

### ✅ 必須做的事：
1. ✅ 建立 **Banner** 和 **Products** 分類
2. ✅ 文章必須勾選對應的分類
3. ✅ 文章必須設定**特色圖片**
4. ✅ 文章狀態必須是「已發佈」

### ❌ 常見錯誤：
1. ❌ 忘記勾選分類
2. ❌ 沒有設定特色圖片
3. ❌ 文章狀態是「草稿」而不是「已發佈」
4. ❌ 分類的 slug 不是小寫（必須是 `banner` 和 `products`）

## 💡 提示

- **文章 = Banner/Product**：不需要建立特殊的內容類型，就是一般的「文章」
- **分類 = 區分用途**：透過分類來區分哪些是 Banner，哪些是 Product
- **特色圖片 = 主要圖片**：這是最重要的，一定要設定！
