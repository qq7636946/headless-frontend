# 🎉 安裝自訂文章類型外掛

## ✅ 已完成

我已經為您建立了一個 WordPress 外掛，可以在左側選單新增：
- 📸 **Banner 輪播**
- 🛋️ **產品展示**

---

## 📦 安裝步驟

### **步驟 1：啟用外掛**

1. 登入 WordPress 後台：`http://localhost/headless/wp-admin`
2. 前往：**外掛 → 已安裝外掛**
3. 找到：**DELO SOFA Custom Post Types**
4. 點擊：**啟用**

---

## 🎯 啟用後會看到

左側選單會出現兩個新選項：

## 📝 使用方式

### **建立 Banner**

1. 點擊左側：**Banner 輪播 → 新增 Banner**
2. 填寫內容：
   ```
   標題：沙發必須與眾不同
   內容：義式輕奢生活，從一張沙發開始
   
   ↓ 往下滾動，會看到「Banner 詳細資訊」區塊 ↓
   
   副標題：Living Room Exceptional
   英文標題：Different, by Design.
   按鈕文字：探索系列
   按鈕連結：/products
   ```
3. 右側設定：**特色圖片**（上傳背景圖）
4. 點擊：**發佈**

### **建立產品**

1. 點擊左側：**產品展示 → 新增產品**
2. 填寫內容：
   ```
   標題：義式家居設計哲學
   內容：21年家具生產製造設計經驗積澱...
   
   ↓ 往下滾動，會看到「產品詳細資訊」區塊 ↓
   
   國家：德國（下拉選單）
   年份：SINCE 1999
   特色描述：聚焦牛皮純手工製成，保留細節紋理...
   ```
3. 右側設定：**特色圖片**（上傳產品圖）
4. 點擊：**發佈**

---

## 🎨 自訂欄位說明

### **Banner 自訂欄位**
- **副標題**：顯示在主標題上方（例如：Living Room Exceptional）
- **英文標題**：顯示在主標題下方（例如：Different, by Design.）
- **按鈕文字**：CTA 按鈕的文字（例如：探索系列）
- **按鈕連結**：按鈕點擊後的連結（例如：/products）

### **產品自訂欄位**
- **國家**：下拉選單（德國、意大利、西班牙、美國、加拿大）
- **年份**：品牌年份（例如：SINCE 1999）
- **特色描述**：產品特色文字

## ✅ 優點

### **使用自訂文章類型的好處：**

1. ✅ **獨立選單**：Banner 和 Product 有自己的選單項目
2. ✅ **專屬欄位**：每種類型都有專屬的自訂欄位
3. ✅ **更清晰**：不會和一般文章混在一起
4. ✅ **更專業**：符合 WordPress 最佳實踐
5. ✅ **REST API**：自動支援 REST API

---

## 🔍 測試 API

啟用外掛後，可以測試以下 API：

- **Banner API**: `http://localhost/headless/wp-json/wp/v2/banner?_embed`
- **Product API**: `http://localhost/headless/wp-json/wp/v2/product?_embed`

## 🆘 如果外掛沒有出現

### **方法 1：檢查檔案位置**
確認檔案在：`d:\wordpress\wordpress\wp-content\plugins\delo-custom-post-types.php`

### **方法 2：重新整理外掛頁面**
在「外掛 → 已安裝外掛」頁面按 `Ctrl + F5` 重新整理

### **方法 3：檢查檔案權限**
確認檔案有讀取權限
