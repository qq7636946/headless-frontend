# 🚀 快速開始：WordPress 後台設定

## 📌 第一步：建立分類

1. 登入 WordPress 後台：`http://localhost/headless/wp-admin`
   - 帳號：`admin`
   - 密碼：`admin123`

2. 前往：**文章 → 分類**

3. 新增以下分類：

### Banner 分類
- **名稱**：Banner
- **代稱（slug）**：banner
- 點擊「新增分類」

### Products 分類
- **名稱**：Products  
- **代稱（slug）**：products
- 點擊「新增分類」

---

## 🎨 第二步：建立 Banner 輪播

1. 前往：**文章 → 新增文章**

2. 填寫內容：
   ```
   標題：沙發必須與眾不同
   內容：（可留空或填寫描述）
   ```

3. **右側欄位**：
   - 勾選分類：**Banner**
   - 點擊「設定特色圖片」→ 上傳客廳沙發圖片

4. 點擊「發佈」

5. **重複以上步驟**，建立 2-3 個 Banner

---

## 🛋️ 第三步：建立產品展示

1. 前往：**文章 → 新增文章**

2. 填寫內容：
   ```
   標題：義式家居設計哲學
   內容：21年家具生產製造設計經驗積澱，摸索推出義式輕奢品牌——德客·皮沙發，專為3億新中產量身打造。
   ```

3. **右側欄位**：
   - 勾選分類：**Products**
   - 點擊「設定特色圖片」→ 上傳沙發產品圖片

4. 點擊「發佈」

5. **重複以上步驟**，建立 3-5 個產品

---

## ✅ 第四步：查看效果

1. 開啟前端網站：`http://localhost:3000`

2. 您應該會看到：
   - ✅ Hero Banner 顯示您建立的 Banner 文章
   - ✅ Orbit Section 顯示您建立的 Product 文章
   - ✅ 最新消息顯示一般文章（帶分類標籤）

---

## 🔍 檢查分類 ID

如果 Banner 或 Product 沒有顯示，可能是分類 ID 不正確：

1. 前往：**文章 → 分類**
2. 點擊 **Banner** 分類
3. 查看網址列：`...?tag_ID=X`（X 就是分類 ID）
4. 記下 Banner 和 Products 的 ID

5. 如果 ID 不是 1 和 2，請修改：
   - 檔案：`d:\wordpress\headless-frontend\lib\wordpress.ts`
   - 找到：`categories=1` 和 `categories=2`
   - 改成實際的分類 ID

## 🆘 常見問題

### Q: Banner 沒有顯示？
A: 確認文章已勾選「Banner」分類，並設定了特色圖片

### Q: 產品沒有顯示？
A: 確認文章已勾選「Products」分類，並設定了特色圖片

### Q: 圖片顯示不正確？
A: 確認已在 WordPress 後台上傳並設定「特色圖片」

### Q: 分類 ID 在哪裡查看？
A: 文章 → 分類 → 點擊分類名稱 → 查看網址列的 `tag_ID=X`
